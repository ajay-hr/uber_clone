## 🌐 Deployment

To deploy this for public testing:

### Using Railway / Render:
1. Connect your GitHub repo.
2. Add a **Redis** service.
3. Set up two services:
   - **Web**: `uvicorn main:app --host 0.0.0.0 --port $PORT`
   - **Worker**: `celery -A app.workers.tasks worker --loglevel=info`
4. Update the `API_URL` in `test_ui.html` to point to your deployed URL.
